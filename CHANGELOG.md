# 更新日誌 (CHANGELOG)

本專案遵循「易用性優先」的原則進行開發。以下是本開發階段的所有變更紀錄：

## [v1.3.0] - 2026-04-27 (當前版本)
### ✨ 極致易用性優化 (Usability Polish)
- **[新增]** 手動深色模式切換按鈕 (Manual Theme Toggle)。
- **[優化]** 語音輸入狀態回饋：點擊麥克風時圖示呈現閃爍動畫。
- **[優化]** UI 交互細節：優化 AI 打字速度與骨架屏顯示時機。
- **[持久化]** 記憶使用者的主題偏好 (Dark/Light Mode persistence)。

## [v1.2.0] - 2026-04-27
### 🧠 核心功能大改版 (Major Overhaul)
- **[功能]** 實作 **LocalStorage** 對話紀錄持久化，重新整理不丟失內容。
- **[功能]** 實作 **Natural Typing** 效果，模擬真人打字產出回應。
- **[功能]** 實作 **Skeleton Loading** 骨架屏，降低感知等待時間。
- **[功能]** 整合 **Web Speech API**，支援中文語音輸入。
- **[功能]** 商品卡片新增一鍵分享/複製連結功能。
- **[導引]** 實作 **Dynamic Placeholder**，輸入框自動輪播功能範例。

## [v1.1.0] - 2026-04-27
### 📱 移動端與 RWD 優化
- **[調整]** 針對螢幕 < 480px 自動切換為「全螢幕 App 模態」。
- **[優化]** 支援 iOS/Android **Safe Area** (瀏海屏與底部手勢區) 適配。
- **[優化]** 採用 `100dvh` 解決行動端瀏覽器工具列遮擋問題。

## [v1.0.1] - 2026-04-27
### 📝 文件與邏輯補全
- **[文件]** 建立「產品需求規格書 (Product Spec)」。
- **[文件]** 建立「開發者交接指南 (Developer Guide)」。
- **[文件]** 新增「釐清問題表 (Discovery Questionnaire)」與「易用性價值分析」。

## [v1.0.0] - 2026-04-27
### 🚀 初始版本發布
- **[核心]** 基於 Vanilla JS/CSS 的智能客服原型。
- **[設計]** 採用現代玻璃擬態 (Glassmorphism) 風格。
- **[邏輯]** 實作三階層意圖引導機制（FAQ -> 猜測引導 -> 真人轉接）。
