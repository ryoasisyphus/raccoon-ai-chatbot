# Raccoon AI: 提示詞工作流 (Prompt Workflow)

本專案旨在利用 AI 提示詞工程快速打造高品質的 AI 客服原型。

## 資料夾結構
- `/prompts`: 存放各階段的系統提示詞。
  - `01_product_scoping.md`: 用於初步需求釐清、規格產出與快速原型實作。

## 使用說明
1. **複製提示詞**：打開 `/prompts/01_product_scoping.md`。
2. **啟動工作流**：將內容貼給 AI 助理（如 Antigravity 或 Claude），它將會引導你完成需求釐清、Spec 產出，並直接產出一個可運行的 Demo。
3. **疊代與優化**：根據輸出的「後續優化建議」進行下一輪的提示詞調整。

## 專案目標
打造一個具備 FAQ、商品推薦、真人轉接功能的 AI 客服，並以「極致易用性」為核心開發原則。
